estimate_parkers_model

 estimate_parkers_model (measured:numpy.ndarray, timeline:numpy.ndarray,
                         return_result:bool=False, verbose:bool=True)

Estimate best Parker parameters for measured data.

Type Default Details
measured ndarray Measured data
timeline ndarray Timeline in seconds
return_result bool False When True, return OptimizeResult, otherwise return fitted parameters
verbose bool True Whether estimated parameters will be printed
Returns typing.Union[scipy.optimize._optimize.OptimizeResult, numpy.ndarray]

parker_cost_function

 parker_cost_function (x:tuple, t:numpy.ndarray, y:numpy.ndarray)

Calculate difference between parker function and given data y.

Type Details
x tuple Parker parameters (A1, A2, T1, T2, sigma1, sigma2, alpha, beta, s, tau)
t ndarray Timeline in seconds
y ndarray Measured data
Returns ndarray Difference

parker

 parker (x:tuple, N:int, t:numpy.ndarray)

Calculate a AIF signal, given the supplied parameters.

Type Details
x tuple Parker parameters (A1, A2, T1, T2, sigma1, sigma2, alpha, beta, s, tau)
N int Length of signal
t ndarray Timeline in seconds (length N)
Returns ndarray Signal at timepoints t 
x = (0.809, 0.33, 0.17046, 0.365, 0.0563, 0.132, 1.050, 0.1685, 38.078, 0.483)
timeline = np.arange(0., 300., 2.5)
p = parker(x, len(timeline), timeline)

fig, ax = plt.subplots()
ax.plot(timeline, p)
ax.set_ylabel('Parker [a.u.]')
ax.set_xlabel('Time [s]')
plt.show()
